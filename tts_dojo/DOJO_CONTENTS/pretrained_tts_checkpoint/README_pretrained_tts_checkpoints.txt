The contents of this folder are managed by the TextyMcSpeechy scripts.  You don't need to copy anything here.
It is expected to contain a pretrained checkpoint file (or a symlink to one in PRETRAINED_CHECKPOINTS)
example filename: epoch=2307-step=558536.ckpt 

checkpoints for common piper voice modules are available at:
https://huggingface.co/datasets/rhasspy/piper-checkpoints
